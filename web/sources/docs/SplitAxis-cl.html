cl

Split axis

Constructor

new cl( )

Mixes In:

Methods

{SplitAxis} all( function callback )

Calls a callback onto each chunk axes. The callback receives two parameters: 1) the ```axis``` itself and 2) the ```index``` of the axis in the stack
Parameters:
Name Type Description
callback function The callback to be applied to each axes
Returns:
{ SplitAxis } The current axis instance

{SplitAxis} fixGridIntervalBasedOnAxis( Number axisIndex )

Fixes the major tick interval of all axes based on the one provided as a parameter
Parameters:
Name Type Description
axisIndex Number The index of the reference axis (starting at 0)
Returns:
{ SplitAxis } The current axis instance

{SplitAxis} splitAxis( )

Splits the axis into chunks at the positions defined as a list of parameters.
Parameters:
Name Type Description
...splits function The positions of axis splitting
Returns:
{ SplitAxis } The current axis instance
Example
{% highlight JavaScript %}axis.splitAxis( 0.2, 0.5, 0.8 ); // Creates 4 chunks (0-20%, 20%-50%, 50%-80%, 80%-100%){% endhighlight %}

{SplitAxis} splitSpread( Boolean bln )

Spreads the chunks of the axis based on the relative interval of each one of them, so that the unit / px is constant for each chunk
Parameters:
Name Type Description
bln Boolean ```true``` to enable the spread, ```false``` otherwise
Returns:
{ SplitAxis } The current axis instance

{SplitAxis} setChunkBoundaries( Array.<(Array|Number)> values )

Defines the boundaries of each chunk in axis unit.
Parameters:
Name Type Description
values Array.<(Array|Number)> An array of either 2-component arrays (from-to) or number (mean)
Returns:
{ SplitAxis } The current axis instance
Examples
{% highlight JavaScript %}axis.setChunkBoundaries( [ [ 12, 20 ], [ 100, 200 ] ] ); // First chunk from 12 to 20, second one from 100 to 200 {% endhighlight %} {% highlight JavaScript %}axis.setChunkBoundaries( [ 12, [ 100, 200 ] ] ); // Second chunk from 100 to 200, first chunk with a mean at 12 and min / max determined by the relative widths of the chunks {% endhighlight %}
