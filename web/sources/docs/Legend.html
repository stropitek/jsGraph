Legend

Legend constructor. You should not call this method directly, but rather use graph.makeLegend

Constructor

new Legend( )

Example
{% highlight JavaScript %}var legend = graph.makeLegend( { backgroundColor: 'rgba(255, 255, 255, 0.8)', frame: true, frameWidth: 1, frameColor: 'black', paddingTop: 10, paddingLeft: 10, paddingBottom: 10, paddingRight: 10, frameRounding: 3, movable: false, shapesToggleable: true, isSerieHideable: true, isSerieSelectable: true } );{% endhighlight %}

setPosition( Position position , String alignToX , String alignToY )

Sets the position of the legend
Parameters:
Name Type Description
position Position the position to set the legend to versus the graph main axes (Graph#getXAxis and Graph#getYAxis)
alignToX String "right" or "left". References the legend right or left boundary using the position parameter
alignToY String "top" or "bottom". References the legend top or bottom boundary using the position parameter
Example
{% highlight JavaScript %}legend.setPosition( { x: 'max', y: '0px' }, 'right', 'top' ); // The rightmost side of the legend will at the maximum value of the axis, and will be positioned at the top{% endhighlight %}
