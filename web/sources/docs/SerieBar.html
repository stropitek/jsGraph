SerieBar

Represents a bar serie. Needs to be used exclusively with a bar axis (AxisXBar). Supports error bars, line color, line width, fill color, fill opacity.

Constructor

new SerieBar( )

Example
{% highlight JavaScript %}graph.newSerie("serieName", { fillColor: 'red', fillOpacity: 0.2 }, "bar" );{% endhighlight %}
