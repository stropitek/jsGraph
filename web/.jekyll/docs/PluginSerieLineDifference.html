{Object|Boolean} computeCrossing( Number x11 , Number y11 , Number x12 , Number y12 , Number x21 , Number y21 , Number y22 , Number y22 )

Finds the crossing point between two vector and returns it, or ```false``` if it is not within the x boundaries
Parameters:
Name Type Description
x11 Number First x point of the first vector
y11 Number First y point of the first vector
x12 Number Second x point of the first vector
y12 Number Second y point of the first vector
x21 Number First x point of the second vector
y21 Number First y point of the second vector
y22 Number Second x point of the second vector
y22 Number Second y point of the second vector
Returns:
{ Object | Boolean } An object containing the crossing point in the following format: ```{ x: xCrossing, y: yCrossing }``` or ```false``` if no crossing point can be found
